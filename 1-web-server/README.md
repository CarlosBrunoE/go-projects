# 1 Web Server

A simple web server written in Go using the `net/http` package.

## Getting Started

### Prerequisites

Make sure you have Go installed on your machine. You can download it from [https://golang.org/dl/](https://golang.org/dl/).

### How to run it

```bash
go run main.go
```
